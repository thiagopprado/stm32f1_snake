## Snake
Version of the classic snake game. It uses 4 mecanic keys connected to the MCU GPIO for character control and a Nokia 5110 LED display.

Keyboard
- PB12 -  Right
- PB13 -  Down
- PB14 -  Left
- PB15 -  Up

Display
- PA0 -  DS
- PA1 -  RST
- PA4 -  CS
- PA5 -  SCLK 1
- PA6 -  MISO 1
- PA7 -  MOSI 1
